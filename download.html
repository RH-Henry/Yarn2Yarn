Important Information Before Downloading

⚠ A Windows security window will appear.
This is expected and happens the first time you run the software.

The window you are going to see

When launching the software for the first time, Windows will display a message saying: “Windows protected your PC” and “Unknown publisher”.

If you trust me, simply press “More info” and “Run anyway”.

Why this window appears

By default, Microsoft Windows blocks or warns about any application that:

  • is not signed with a very specific (and expensive) certificate
  • or does not yet have enough “reputation” in Microsoft’s ecosystem

This software is developed by an independent developer, not a large corporation paying for privileged treatment.

Important:
This warning does not mean the software is dangerous, malicious, or infected. It only means that Microsoft cannot automatically vouch for the publisher unless a costly validation scheme is used.

Why this warning cannot simply be “removed”

To make this warning never appear, Microsoft requires a certificate called EV (Extended Validation).

  • Cost: approximately $400–$600 per year
  • Recurring annual payment
  • Issued by a small number of private certification authorities

In practice, this certificate exists primarily to remove this screen. It does not magically make the software more secure.

Let’s be honest:
This is a pay-to-remove-friction system. Independent developers are forced to pay a recurring fee to avoid scaring users, while large companies absorb the cost without blinking.

This is not about security. It is about control and monetization of trust by Microsoft and a handful of certificate authorities.
